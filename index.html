Project Overview

This project focuses on migrating the Adaptive Learning Codebase from the legacy Autogen framework to CrewAI. The goal is to modernize the system by replacing existing agents with CrewAI equivalents, ensuring improved scalability, maintainability, and extensibility.

The migration process includes analyzing current Autogen agents, mapping their roles to CrewAI task-agent models, and implementing workflows for math, language, and career path learning. Comprehensive testing and validation are performed to benchmark CrewAI agent performance against the legacy system. Finally, updated documentation and a detailed migration guide are provided to support future development and onboarding.

Project Tasks

  • Task1: Analyze Existing Autogen Agents
  • Task2: Understand CrewAI Framework
  • Task3: Implement CrewAI Agents (Math, Telugu, Language, Career Path Workflows)
  • Task4: Testing & Validation
  • Task5: Documentation & Handoff
